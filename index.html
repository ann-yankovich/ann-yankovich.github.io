• The Single Responsibility Principle

    Bad example: 

    
    class User {
        setName() {
            this.name = name;
            alert(this.name);
        }
    }

    Good example: 

    
    class User {
        setName(name) {
            this.name = name;
        }
        showName() {
            alert(this.name);
        }
    }

  • The Open Closed Principle

    Bad example: 

    
    class User {
        setName(firstName, lastName) {
            this.name = `${firstName} ${lastName}`;
        }
        showName() {
            alert(this.name);
        }
    }

    Good example: 

    
    class User {
        setName(name) {
            this.name = name;
        }
        setFullName(firstName, lastName) {
            this.fullName = `${firstName} ${lastName}`;
        }
        showName() {
            alert(this.name);
        }
    }

  • The Interface Segregation Principle

    Bad example: 

    
    class User {
        setName(name) {
            this.name = name;
        }
        setFullName(firstName, lastName) {
            this.fullName = `${firstName} ${lastName}`;
        }
        showName() {
            alert(this.name);
        }
    }

    Good example: 

    
    class setUserData {
        setName(name) {
            this.name = name;
        }
        setFullName(firstName, lastName) {
            this.fullName = `${firstName} ${lastName}`;
        }
    }

    class showUserData {
        showName() {
            alert(this.name);
        }
    }
